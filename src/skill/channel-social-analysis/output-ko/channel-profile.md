# 🧠 채널 프로필: 현대 사회 분석

## 1. 채널 포지셔닝 (Channel Positioning)

### 미션
현대 경제-사회 현상 이면에 숨겨진 법칙을 해독하여, 시청자가 **표면을 꿰뚫어** 개인 삶의 현명한 판단을 내릴 수 있도록 돕는다.

### 채널 매니페스토 (Manifesto)
> *"정보는 넘쳐나지만 이해는 부족한 시대에, 우리는 진실의 편에 서기로 했습니다 — 두려움을 주기 위해서가 아니라, 제때 깨어나기 위해서."*

### 콘텐츠 필러 (Content Pillars)

| Pillar | 설명 | 콘텐츠 비중 |
|--------|------|-------------|
| 🔥 **경제 해부** | 경제 구조, 인플레이션, 부동산, 노동 시장 분석 | 35% |
| 🧬 **사회 심리** | 현대 심리 함정, 도파민, 디지털 고독, 세대 압박 | 25% |
| ⚙️ **시스템 & 구조** | 양극화, 도시화, 교육, 의료 | 25% |
| 🔮 **전망 & 전략** | 미래 시나리오, 개인 전략, 저항적 사고 | 15% |

## 2. 타겟 오디언스 (Target Audience)

### 주요 시청자 페르소나 (Primary Persona)

| 특성 | 설명 |
|------|------|
| **페르소나명** | "답을 찾는 사람" |
| **연령** | 20~38세 (MZ세대) |
| **성별** | 무관, 남성 비중 55-60% |
| **학력** | 대졸 이상, 또는 독학 전문가 |
| **소득** | 중위 및 중하위 (월 200만~450만 원) |
| **심리** | '모든 게 비싸지는데 월급은 제자리'라는 느낌; 변화가 필요하다는 걸 알지만 어디서 시작할지 모름 |
| **디지털 행동** | 유튜브/틱톡 매일 시청, 경제 뉴스 소비, 딥다이브 콘텐츠 선호 |
| **고충 (Pain points)** | 내 집 마련 압박, 신용카드 빚, 사회적 비교, 커리어 방향 상실 |
| **열망** | 경제적 자유, 시스템 작동 원리 이해, '쳇바퀴' 탈출 |

### 보조 시청자 페르소나 (Secondary Persona)

| 특성 | 설명 |
|------|------|
| **페르소나명** | "조용한 관찰자" |
| **연령** | 35~50세 |
| **심리** | 경험이 풍부하며, 자신이 관찰한 바를 확인하고 싶어 함 |
| **행동** | 긴 영상 시청, 댓글 읽기, 인터랙션은 적지만 충성도 높음 |

## 3. 톤 & 스타일 가이드 (Tone & Style Guide)

### 메인 톤: **"냉철한 형"**

이 채널은 세상을 겪어본 형처럼 말합니다 — 날카롭고, 아부하지 않지만, 항상 듣는 사람이 더 나아지길 바라는.

| 요소 | ✅ 해야 할 것 | ❌ 하지 말아야 할 것 |
|------|-------------|-------------------|
| **자신감** | 확고하게 주장, 근거 제시 | 두리뭉실, '아마', '혹시' 남발 |
| **극적 분위기** | 통제된 긴장감, 수사학적 질문 | 근거 없는 공포 조성, 저급 낚시 |
| **감정** | 시청자의 고통에 진정한 공감 | 동정, 거만함, '설교' |
| **데이터** | 구체적 출처 인용, 숫자를 이미지로 전환 | 출처 없는 숫자, 모호한 출처 |
| **해결책** | 실용적, 즉시 행동 가능 | 이론적, 현실과 동떨어진 해결책 |
| **언어** | 강력한 은유, 생생한 이미지, 일상적이지만 날카로움 | 학술적 건조함, 과도한 유행어 |

### 감정 곡선 (Emotional Arc) — 매 영상

```
감정
  ▲
  │     ╱╲        ╱╲
  │    ╱  ╲      ╱  ╲        ╱─── 희망
  │   ╱    ╲    ╱    ╲      ╱
  │──╱──────╲──╱──────╲────╱───
  │ 충격     불안    분노   공감    각성
  │
  └──────────────────────────────▶ 시간
   Hook    문제 1  문제 2  해결책
```

### 시그니처 문구 (Signature Phrases)

- "그리고 여기서부터가 진짜 무서운 부분입니다..."
- "이 숫자를 보세요 — [X]에 대한 생각이 완전히 바뀔 겁니다"
- "아무도 이걸 알려주지 않았지만..."
- "이건 여러분의 잘못이 아닙니다. 하지만 여러분의 책임입니다."
- "아직 [X]라고 믿고 있다면, 규칙이 이미 오래전에 바뀐 게임을 하고 있는 겁니다."
- "문제는 '이것이 일어날까?'가 아닙니다 — '언제?'입니다."
- "넘기기 전에, 자신에게 하나만 물어보세요..."

## 4. 언어 정체성 시스템 (Verbal Identity)

### 명명 규칙

| 요소 | 규칙 | 예시 |
|------|------|------|
| **영상 제목** | [충격 수치] + [역설] + [해독 약속] | "소득 40% 증가했는데 67% 청년이 집 못 산다 — 보이지 않는 폭풍은?" |
| **콘텐츠 섹션** | '암호', '폭풍', '안개층'으로 명명 | "두 번째 폭풍: 값싼 도파민과 탈진 세대" |
| **자체 용어** | 브랜드 전용 용어 생성 | "열차 추격 증후군", "엘리베이터 함정", "비행 시대의 보행자" |

### 호칭 체계

- **시청자에게**: "여러분" (친근하면서도 존중 — 존댓말)
- **자칭**: "우리" (같은 편이라는 느낌)
- **악역**: "시스템", "구조", "게임" (특정 개인을 탓하지 않음)

## 5. 콘텐츠 전략 (Content Strategy)

### 게시 일정 (제안)

| 콘텐츠 유형 | 빈도 | 길이 | 플랫폼 |
|-------------|------|------|--------|
| 심층 분석 영상 | 주 1-2회 | 12~20분 | YouTube |
| 포스트 카드 인사이트 | 주 3-4회 | — | 인스타그램, 네이버 블로그, 트위터/X |
| 쇼츠 (영상 발췌) | 주 3-5회 | 30~90초 | TikTok, YouTube Shorts, Reels |

### 콘텐츠 시리즈 (제안)

1. **"[연도] 해부"** — 연간 경제-사회 종합 분석
2. **"사고의 함정"** — 청년을 불이익에 빠뜨리는 인지 오류 시리즈
3. **"세대 보고서"** — 세대 간 비교 (베이비부머, X세대, 밀레니얼, Z세대)
4. **"돈의 지도"** — 사회 내 돈의 흐름 분석
5. **"디지털 각성"** — 기술과 인간 심리의 관계

## 6. 대본 품질 체크리스트 (Script Quality Checklist)

대본을 완성하기 전에 반드시 확인:

- [ ] **Hook < 30초** — 충격이나 호기심 유발
- [ ] 최소 **3개의 수치** — 신뢰할 수 있는 출처
- [ ] 최소 **3개의 강력한 은유** — 추상적 개념을 이미지로 전환
- [ ] **감정 곡선** — 충격 → 불안 → 분노/공감 → 희망
- [ ] 최소 **2개의 시청자 상호작용 질문**
- [ ] **구체적인 실행 해결책**으로 마무리 (이론만이 아닌)
- [ ] **명확한 CTA** (구독, 댓글, 공유)
- [ ] **불필요한 문장 없음** — 모든 문장이 목적에 봉사
- [ ] **설교하지 않음** — 항상 동행, 판단하지 않기
- [ ] 영상 제목에 **수치 + 역설 + 약속** 포함
