# ⚡ 파워 워드 & 강력한 은유

## 1. 파워 워드 — 클릭 유도 키워드

시청자의 심리를 자극하여 CTR과 유지율을 높이는 단어/구문.

### 그룹: 긴급 & 경고
| 단어/구문 | 사용 시점 | 사용 방법 |
|-----------|----------|-----------|
| 잔인한 진실 | 충격적 수치 공개 시 | "이것이 잔인한 진실입니다..." |
| 이중 함정 | 2가지 요소가 동시에 타격할 때 | "청년들은 이중 함정에 빠져있습니다..." |
| 보이지 않는 폭풍 | 아직 폭발하지 않았지만 곧 터질 문제 | "3개의 폭풍이 형성 중입니다..." |
| 블랙홀 | 모든 자원을 빨아들이는 현상 | "부동산이 블랙홀이 되었습니다..." |
| 적색 신호 | 위험 징후 경고 시 | "모두가 무시하는 적색 신호입니다" |
| 돌이킬 수 없는 지점 | 임계점/데드라인 언급 시 | "우리는 돌이킬 수 없는 지점에 가까워지고 있습니다" |
| 시한폭탄 | 문제가 축적 중일 때 | "이것은 시한폭탄입니다..." |
| 도미노 효과 | 연쇄 반응 일으키는 문제 | "도미노 효과가 시작됩니다..." |

### 그룹: 미스터리 & 발견
| 단어/구문 | 사용 시점 | 사용 방법 |
|-----------|----------|-----------|
| 암호 | 숨겨진 메커니즘 설명 시 | "부의 이면에 숨겨진 암호..." |
| 안개층 | 층을 벗겨내며 분석할 때 | "안개층 아래에는..." |
| 보이지 않는 규칙 | 숨겨진 시스템 언급 시 | "아무도 가르쳐주지 않는 규칙" |
| 비밀 지도 | 새로운 가이드 제공 시 | "이 지도가 여러분의 생각을 바꿀 겁니다..." |
| 소스코드 | 근본 원인 분석 시 | "문제의 소스코드를 들여다보겠습니다" |
| 암묵적 진실 | 은폐된 것을 밝힐 때 | "노동 시장의 암묵적 진실" |

### 그룹: 힘 & 행동
| 단어/구문 | 사용 시점 | 사용 방법 |
|-----------|----------|-----------|
| 최강의 무기 | 가장 강력한 해결책 언급 시 | "여러분의 최강의 무기는 각성입니다" |
| 각성 | 인식 전환 촉구 시 | "제때의 각성, 그것이 힘입니다" |
| 저항 전략 | 역행하는 해결책 제시 시 | "이것은 저항 전략입니다..." |
| 나침반 | 삶의 원칙 제시 시 | "다가올 10년의 나침반입니다" |
| 지렛대 | 힘을 배가하는 방법 언급 시 | "기술이야말로 여러분의 지렛대입니다" |
| 정신 백신 | 심리 방어법 언급 시 | "이것을 정신 백신이라 생각하세요" |

### 그룹: 감정 & 공감
| 단어/구문 | 사용 시점 | 사용 방법 |
|-----------|----------|-----------|
| 여러분의 잘못이 아닙니다 | 감정 확인 시 | "이 피로는 여러분의 잘못이 아닙니다" |
| 쳇바퀴 인생 | 무의미한 반복 묘사 시 | "쳇바퀴 인생에 갇혀 있는 겁니다..." |
| 탈진 세대 | 청년 세대 언급 시 | "탈진 세대는 게으른 게 아닙니다" |
| 조용한 고통 | 아무도 말하지 않는 문제 묘사 시 | "이것이 조용한 고통입니다..." |
| 소리 없는 비명 | 무력감 표현 시 | "중산층의 소리 없는 비명" |

## 2. 은유 시스템 (Metaphor System)

### 경제 — 경기장 & 게임 은유

| 원개념 | 은유 | 대본 예시 |
|--------|------|-----------|
| 인플레이션이 저축을 갉아먹음 | 흰개미가 돈을 먹다 | "인플레이션은 흰개미입니다 — 눈에 보이지 않지만 매년 재정의 집을 5%씩 갉아먹습니다" |
| 집값이 월급보다 빠르게 오름 | 자전거로 KTX를 쫓다 | "있는 힘껏 페달을 밟지만, 아파트 가격은 이미 KTX 속도로 달립니다" |
| 부채의 소용돌이 | 모래 늪 | "신용카드 빚은 모래 늪과 같습니다 — 발버둥칠수록 더 깊이 빠집니다" |
| 자산 불평등 | 엘리베이터와 계단 | "부자는 엘리베이터, 서민은 계단 — 그리고 계단은 갈수록 길어집니다" |
| 화폐가치 하락 | 녹는 얼음 | "현금은 한여름 얼음처럼 녹고 있습니다" |
| 부동산 투기 | 의자 빼기 게임 | "부동산은 의자 빼기 게임이 되었습니다 — 음악이 멈출 때 의자 없는 사람은 추락합니다" |

### 심리 — 생물학 & 자연 은유

| 원개념 | 은유 | 대본 예시 |
|--------|------|-----------|
| 소셜미디어 중독 | 값싼 도파민 | "틱톡 스와이프 한 번은 값싼 도파민 한 방울 — 3초 기쁨, 3시간 공허" |
| 번아웃 | 배터리 방전 | "게으른 게 아닙니다 — 2015년에 충전한 배터리로 2025년 소프트웨어를 돌리고 있을 뿐입니다" |
| 사회적 비교 | 거울 미로 | "인스타그램은 거울 미로입니다 — 수백 개의 '완벽한' 버전이 보이고, 모든 거울이 왜곡합니다" |
| 디지털 고독 | 대양 속 사막 | "연결의 바다에서 헤엄치지만, 고독의 사막에서 목마릅니다" |
| FOMO | 브라우저 탭 50개 | "FOMO는 인생을 웹 브라우저로 만듭니다 — 탭 50개 열었지만 하나도 제대로 못 읽습니다" |
| 성공 압박 | 끝없는 트랙 | "사회가 그린 트랙에는 결승선이 없습니다 — 언제 멈춰도 되는지 아무도 모릅니다" |

### 시스템 — 건축 & 기계 은유

| 원개념 | 은유 | 대본 예시 |
|--------|------|-----------|
| 낙후된 교육 시스템 | 20세기 공장 | "학교는 여전히 산업시대 공장입니다 — 이미 멈춘 생산라인을 위한 인력을 대량 생산합니다" |
| 빈부 격차 | 2층 도시 | "우리는 2층 도시에 삽니다 — 위층은 하늘 전체가 보이고, 아래층은 천장만 보입니다" |
| 고령화 | 뒤집히는 삼각형 | "인구 피라미드가 뒤집히고 있습니다 — 더 적은 청년이 더 많은 노인을 떠받치다, 결국 무너집니다" |
| 과도한 도시화 | 도시 블랙홀 | "대도시는 블랙홀처럼 농촌의 모든 자원, 인재, 기회를 빨아들입니다" |
| 기후변화 | 끓는 물 속 개구리 | "우리는 서서히 뜨거워지는 물 속의 개구리입니다 — 충분히 느려서 뛰쳐나오지 못합니다" |

## 3. 브랜드 전용 용어 (Brand Terminology)

채널만의 언어 정체성을 만드는 자체 용어:

| 용어 | 정의 | 사용 시점 |
|------|------|-----------|
| **열차 추격 증후군** | 노력은 늘어나지만 목표와의 거리는 벌어짐 | 소득 vs. 자산가격 |
| **엘리베이터 함정** | 시스템이 나를 올려줄 거라는 믿음, 하지만 엘리베이터는 고장 | 잘못된 사회적 기대 |
| **재정 허무주의** | '집 못 사니까 쓰고 살자, 오늘을 즐기자'는 심리 | 청년 소비 심리 |
| **값싼 도파민** | 소셜미디어의 즉각적 쾌감이 실질적 성취를 대체 | 기술 중독 |
| **도시 블랙홀** | 대도시가 모든 자원을 빨아들여 다른 지역을 사막화 | 도시화 |
| **비행 시대의 보행자** | 구시대 도구/사고로 새 세상을 살아가는 사람 | 역량 부적합 |
| **보이지 않는 경제** | 프리랜서, 긱 이코노미 등 측정 불가능한 노동 | 실질 노동시장 |
| **샌드위치 세대** | 부모도 부양하고 자기 미래도 책임져야 하는 청년 | 다중 재정 압박 |
| **꿈의 인플레이션** | 꿈의 실현 비용 상승이 소득 상승보다 빠름 | 열망과 현실의 괴리 |
| **레벨 99 게임의 레벨 1 플레이어** | 기본 장비로 초고난도 세상에 진입하는 청년 | 세대 격차 |
| **비교 미로** | 소셜미디어에서의 비교가 만드는 불행 | 비교 심리 |

## 4. 대비 쌍 (Contrast Pairs)

대본에서 강력한 역설을 만들기 위한 쌍:

| 대비 쌍 | 사용 예시 |
|---------|-----------|
| 명목 소득 ↔ 실질 구매력 | "소득은 역대 최고인데 구매력은 역대 최저" |
| 연결 ↔ 고독 | "50억 명이 온라인이지만 역사상 가장 외로운 시대" |
| 기회 ↔ 불평등 | "기회는 더 많아졌지만, 이미 정상에 있는 사람들에게만" |
| 자유 ↔ 불안 | "가장 자유롭지만 — 가장 불안하기도" |
| 국가 부 ↔ 개인 빈곤 | "GDP는 오르는데 지갑은 여전히 얇다" |
| 가장 젊음 ↔ 가장 지침 | "사무실에서 가장 젊지만 가장 지쳐있는 세대" |
| 가장 똑똑함 ↔ 가장 혼란 | "지식 접근성은 최고인데 혼란은 최고인 세대" |
| 빠름 ↔ 길 잃음 | "그 어느 때보다 빠르게 달리지만, 그 어느 때보다 멀리 길을 잃었다" |

## 5. 전환 구문 (Transition Phrases)

### 심화/깊이 들어가기
- "하지만 이건 빙산의 일각에 불과합니다..."
- "한 층 더 파고 들면..."
- "다음 부분이야말로 저를 잠 못 들게 했습니다..."
- "한 발짝 물러서서 전체 그림을 보겠습니다..."

### 문제에서 해결책으로 전환
- "이 영상은 두려움을 주려는 것이 아닙니다. 각성시키려는 것입니다."
- "나쁜 소식은 충분합니다. 이제 대응법을 이야기하겠습니다."
- "문제를 아는 것이 1단계입니다. 하지만 2단계가 게임 체인저입니다."

### 시청자 참여 유도
- "댓글로 알려주세요: 여러분은 어떤 단계에 있나요?"
- "3초만 멈추세요. 스스로에게 물어보세요: [질문]. 그리고 계속하세요."
- "혹시 [X]를 느낀 적 있나요? 있다면, 혼자가 아닙니다."

## 6. 영상 제목 — 파워 타이틀 공식

### 제목 공식

```
[충격 수치] + [역설] + [해독 약속]
```

### 제목 예시

- "소득 40% 올랐는데 67% 청년이 집 못 산다 — 3개의 보이지 않는 폭풍"
- "가장 똑똑한 세대가 가장 지친 이유 | 열차 추격 증후군 해부"
- "50억 ONLINE인데 역사상 가장 외롭다 — 디지털 고독의 암호"
- "월급 350만 원, 37년 저축해도 집 못 산다 — 답 없는 방정식"
- "2030년: 40% 직업이 사라진다 — 준비했나요?"
- "부모보다 잘 살고 있나요? 이 숫자가 답을 바꿀 겁니다"

### 제목 작성 규칙
1. **최대 80자** (모바일에서 완전 표시)
2. **1-2개 단어 대문자** 강조
3. **구체적 수치** 제목 앞부분에 배치
4. **호기심 갭** (curiosity gap) 생성
5. **저급 낚시 단어 지양**: "충격", "경악" → 신뢰감 있는 표현으로 대체
