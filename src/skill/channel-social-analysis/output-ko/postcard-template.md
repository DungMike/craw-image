# 🎴 포스트 카드 템플릿

## 1. 개요

포스트 카드는 인스타그램, 트위터/X, 네이버 블로그, 틱톡용 정적 이미지 또는 캐러셀입니다.
- **신규 유입**: 소셜미디어에서 유튜브로 유도
- **콘텐츠 요약**: 긴 영상의 핵심 인사이트 응축
- **토론 유도**: 댓글과 공유 촉진
- **브랜드 구축**: 언어/비주얼 일관성 유지

---

## 2. 포스트 카드 유형

### 유형 1: 인용 카드 (Quote Card)

**목적**: 영상에서 강력한 인용구 1개 + 짧은 맥락.

**레이아웃**:
```
┌─────────────────────────────────┐
│  [채널 로고 — 상단 모서리]       │
│     "영상에서 가장              │
│      강력한 인용구"              │
│     — 짧은 맥락 1줄             │
│  [영상 제목 — 하단, 작게]       │
│  [CTA: YouTube에서 전체 보기]   │
└─────────────────────────────────┘
```

**디자인 규칙**: 다크 배경, Bold 산세리프 36-48px, 화이트/골드(#F0C040), 1080x1080px(인스타) 또는 1200x630px(트위터)

**카피**: 인용 최대 25자, 캡션 2-3문장 + 해시태그 + CTA

**예시**:
> 인용: *"저축이 아닙니다 — 안심료를 내고 있는 겁니다."*
> 맥락: *인플레이션 3.5%인데 예금 금리 2.5%일 때*

---

### 유형 2: 수치 폭탄 카드 (Data Bomb Card)

**목적**: 충격적 수치 1개 + 맥락.

**레이아웃**:
```
┌─────────────────────────────────┐
│         [큰 수치] 67%           │
│   30세 미만 청년 중             │
│   내 집 마련 불가능 비율         │
│   ─────────────────────         │
│   출처: 한국은행                 │
│  [로고]    [전체 영상 보기]     │
└─────────────────────────────────┘
```

**디자인 규칙**: 수치 96-120px Bold 엑센트컬러, 설명 20-24px 라이트, 출처 14px 이탤릭 70% 투명도

---

### 유형 3: 캐러셀 (3-7장)

**목적**: 영상 내용을 인사이트별로 요약.

**디자인 규칙**: 최대 7페이지, 각 페이지 1가지 아이디어, 1080x1350px(4:5), 통일된 컬러/폰트/레이아웃

---

### 유형 4: 대비 카드 (Versus Card)

**목적**: 대립되는 두 현실 비교로 역설 생성.

**디자인 규칙**: 명확한 양분, 대칭 폰트, 하단에 연결 질문

---

### 유형 5: 스토리 카드 (Thread/Story Card)

**목적**: 인스타/트위터 스토리용. 1080x1920px(9:16)

## 3. 컬러 팔레트 & 타이포그래피

### 컬러

| 이름 | Hex | 용도 |
|------|-----|------|
| **Deep Black** | `#0A0A0A` | 메인 배경 |
| **Dark Navy** | `#0D1117` | 보조 배경 |
| **Ghost White** | `#E8E8E8` | 메인 텍스트 |
| **Signal Gold** | `#F0C040` | 수치/인용 강조 |
| **Alert Red** | `#E53E3E` | 경고 |
| **Hope Teal** | `#38B2AC` | 해결책 |
| **Subtle Gray** | `#6B7280` | 보조 텍스트 |

### 그라데이션

| 그라데이션 | 용도 |
|-----------|------|
| `#0A0A0A → #1A1A2E` | 일반 카드 |

| `#F0C040 → #E53E3E` | 위급 엑센트 |
| `#38B2AC → #2D9CDB` | 희망 엑센트 |

### 타이포그래피

| 역할 | 폰트 | 굵기 | 크기 |
|------|------|------|------|
| 메인 제목 | Pretendard / Noto Sans KR | Bold (700) | 36-48px |
| 큰 수치 | Pretendard / Montserrat | Extra Bold (800) | 72-120px |
| 내용 | Pretendard / Noto Sans KR | Regular (400) | 18-24px |
| 출처 | Pretendard | Light (300) | 14-16px |
| CTA | Pretendard | Semi Bold (600) | 18-20px |

## 4. 제작 프로세스

1. **추출** — 대본에서 인사이트 3-5개 선정
2. **유형 선택** — 각 인사이트에 적합한 카드 유형
3. **카피 작성**: 인용 25자, 데이터 숫자+1문장, 캡션 2-3문장+해시태그
4. **디자인** — 템플릿/컬러 팔레트 적용
5. **리뷰** — 체크리스트 확인
6. **스케줄링** — 콘텐츠 캘린더 등록

### 게시 일정 (영상 1개 = 카드 3-5개)

| 시점 | 카드 | 플랫폼 | 비고 |
|------|------|--------|------|
| 영상 1일 전 | 티저 카드 | 전체 | 궁금증 유발 |
| 게시일 | 수치 폭탄 카드 | 전체 | 트래픽 유입 |
| +1일 | 캐러셀 | 인스타그램 | 미시청자 요약 |
| +2일 | 대비 카드 | 트위터/네이버 | 토론 유도 |
| +3일 | 인용 카드 | 전체 | 사이클 마무리 |

## 5. 체크리스트

- [ ] **3초 안에 읽기 가능** — 즉각적으로 명확한 메시지
- [ ] **충분한 대비** — 텍스트가 배경 위에 뚜렷함
- [ ] **채널 로고** 포함 (작게)
- [ ] **명확한 CTA** — 다음 행동 안내
- [ ] **정확한 크기** — 플랫폼 기준
- [ ] **브랜드 일관성** — 같은 컬러, 폰트, 스타일
- [ ] **캡션** + 해시태그 완료
- [ ] **맞춤법 오류 없음**
- [ ] **세이프 존** — 중요 내용이 가장자리에서 잘리지 않음
