## 2. 썸네일 (Thumbnail)

### 효과적인 썸네일 공식

```
┌──────────────────────────────────────┐
│  [큰 수치,       [감정 전달          │
│   골드/레드]      이미지/아이콘]      │
│  [보조 텍스트 — 호기심 유발 3-5자]   │
│            [로고 작게 모서리]        │
└──────────────────────────────────────┘
```

### 썸네일 규칙

| 요소 | 규칙 |
|------|------|
| 크기 | 1920x1080px (16:9) |
| 수치 | 크게, Bold, 대비 색상 (어두운 배경에 골드/레드) |
| 텍스트 | 최대 6-8자 |
| 폰트 | Pretendard / Montserrat Bold |
| 이미지 | 관련 아이콘/일러스트 + 강한 표정 |
| 배경 | 다크, 그라데이션, 또는 블러 이미지 |
| 대비 | 최소 4.5:1 (WCAG AA) |
| 브랜딩 | 채널 로고, 우하단 |

### 썸네일 텍스트 예시

| 주제 | 텍스트 | 수치 |
|------|--------|------|
| 소득 vs. 주거 | "37년 모아도 못 산다?" | 67% |
| 디지털 고독 | "50억 ONLINE 여전히 외롭다" | 📱→💔 |
| 청년 번아웃 | "지쳤는데 멈출 수 없다" | 72% |
| 노동시장 미래 | "40% 직업이 사라진다" | 2030 |

## 3. 영상 이미지 구조

### 파트별 이미지 분류

| 파트 | 이미지 유형 | 쇼트 당 길이 |
|------|------------|-------------|
| **Hook** | 키네틱 타이포그래피 + 어두운 스톡 | 3-5초 |
| **Bridge** | 애니메이션 인포그래픽 + 은유 이미지 | 5-8초 |
| **Storm 1-3** | 데이터 비주얼라이제이션 + B-roll | 4-7초 |
| **Mirror** | 미니멀 + 인물 요소 | 6-10초 |
| **Blueprint** | 아이콘 애니메이션 + 체크리스트 | 4-6초 |
| **Close** | 브랜딩 + CTA 요소 | 5-8초 |

### 비주얼 스타일 가이드

| 요소 | 규칙 |
|------|------|
| **톤** | 쿨 톤: 네이비, 그레이, 다크 / 웜 엑센트: 골드, 오렌지 강조용 |
| **모션** | 부드럽고 느리게 — 끊김 없이, 과도한 전환 지양 |
| **화면 타이포** | 중요 인용/수치를 화면에 표시 |
| **출처 자막** | 수치 언급 시 출처명 하단 삼분할에 표시 |
| **전환** | 페이드, 스무스 줌, 약한 글리치 (폭풍 사이) |
| **텍스처** | 약한 필름 그레인 오버레이 — 다큐멘터리 느낌 |

## 4. 화면 타이포그래피 효과

| 종류 | 사용 시점 | 효과 | 예시 |
|------|----------|------|------|
| **Number Pop** | 중요 수치 | 큰 숫자 줌인 + 광택 | 67% (골드, 120px) |
| **Quote Flash** | 강력한 인용 | 페이드인 (한 단어씩 또는 전체) | "여러분의 잘못이 아닙니다" |
| **Source Tag** | 출처 인용 | 하단 삼분할 슬라이드인 | "출처: 통계청, 2025" |
| **Chapter Title** | 폭풍 이름 | 풀스크린, 볼드, 페이드 | "두 번째 폭풍" |
| **Term Define** | 자체 용어 | 박스 하이라이트 + 의미 | "열차 추격 증후군: ..." |
| **Question Pop** | 시청자 질문 | 텍스트 + 질문 아이콘 | "몇 년이 필요한가요?" |

## 5. 데이터 비주얼라이제이션

### 데이터별 차트 유형

| 데이터 유형 | 차트 | 스타일 |
|------------|------|--------|
| 2개 값 비교 | 대칭 막대 차트 | 다크 배경, 골드 vs 레드 |
| 시간 추이 | 라인 차트 (애니) | 다크 배경에 골드 라인 |
| 퍼센트 비율 | 도넛/링 차트 | 강조 부분만 밝게 |
| 세대 비교 | 누적 막대/타임라인 | 세대별 다른 컬러 |

### 데이터 비주얼 규칙

1. **미니멀**: 필요한 데이터만, 장식 배제
2. **애니메이션**: 차트가 점차 그려지는 방식 (draw-on)
3. **하이라이트**: 주목할 부분만 밝게, 나머지 흐리게
4. **라벨 명확**: 수치 + 단위 + 출처 표시
5. **길이**: 차트 당 5-8초

          │ 인포그래픽 소개

          │ B-roll 예시

          │ 분할 비교

## 7. 후반작업

### 1단계: Assembly Cut
1. 오디오 트랙 (TTS + BGM + SFX) 배치
2. 타임라인에 맞춰 이미지/영상 배치
3. 나레이션과 비주얼 싱크 (주요 아이디어 = 1 비주얼 교체)

### 2단계: Rough Cut
1. 키네틱 타이포그래피 추가
2. 데이터 비주얼 애니메이션 추가
3. 전환 효과 추가
4. 페이싱 확인: 새 이미지 없이 **2초 이상 공백 없음**

### 3단계: Fine Cut
1. 타이포그래피 타이밍 미세 조정
2. 필름 그레인 오버레이 추가 (투명도 5-10%)
3. 컬러 그레이딩: 쿨 톤, 높은 대비, 골드 하이라이트
4. 약한 비네팅 (가장자리 어둡게)

### 4단계: Sound Mix
1. 밸런스: 보이스 -6dB, BGM -18dB, SFX -12dB
2. 덕킹: 보이스 나올 때 BGM 자동 감소
3. 마스터: -14 LUFS (유튜브 기준)

### 5단계: Export
| 설정 | 값 |
|------|-----|
| 해상도 | 1920x1080 또는 3840x2160 (4K) |
| 프레임률 | 30fps 또는 60fps |
| 코덱 | H.264 (YouTube) / H.265 (4K) |
| 비트레이트 | 16-24 Mbps (1080p) / 45-68 Mbps (4K) |
| 오디오 | AAC 320kbps, 48kHz |

## 8. YouTube SEO & 메타데이터

### 제목
**공식**: `[충격 수치] + [역설] — [해독 약속]`
- 최대 60자, 1-2개 단어 강조, 구체적 수치, 호기심 갭

### 설명 (Description)

```
[Hook 2-3문장 — 핵심 역설 요약]

📊 이 영상에서:
0:00 - 오프닝
0:45 - 배경
2:00 - [폭풍 1 이름]
5:30 - [폭풍 2 이름]
9:30 - [폭풍 3 이름]
13:30 - 반영
15:00 - 3가지 행동 전략
16:30 - 결론

📚 참고 자료:
• [출처 1] — [링크]
• [출처 2] — [링크]

🔔 채널 구독: [링크]
📱 팔로우: [인스타그램] | [트위터/X] | [틱톡]

💬 댓글: [영상 내 CTA 질문]

#[해시태그1] #[해시태그2] #[해시태그3]
```

### 태그

| 태그 유형 | 예시 |
|----------|------|
| 핵심 주제 | 사회분석, 한국경제, 재테크 |
| 롱테일 | 왜 청년이 지쳤나, 서울 아파트 가격 2025 |
| 트렌딩 | MZ세대 한국, 인플레이션 2025, 부동산 |
| 브랜드 | [채널명], 해독, 보이지 않는 폭풍 |

### 엔드 스크린 (마지막 20초)

| 위치 | 요소 | 링크 대상 |
|------|------|-----------|
| 왼쪽 | 다음 영상 (같은 시리즈) | 다음 영상 |
| 오른쪽 | 채널 인기 영상 | 최고 성과 영상 |
| 중앙 하단 | 구독 버튼 | 채널 |

## 9. 업로드 & 크로스 프로모션

### 업로드 일정

| 시점 | 행동 | 플랫폼 |
|------|------|--------|
| T-1 | 티저 포스트 카드 | 인스타, 트위터 |
| T0 (게시일) | 영상 업로드 — 19:00-20:00 KST | YouTube |
| T0 + 2h | 수치 폭탄 카드 + 영상 링크 | 인스타, 트위터 |
| T+1 | 쇼츠 (가장 매력적 60초) | YouTube Shorts |
| T+1 | 릴스/틱톡 (30-90초) | 인스타 릴스, 틱톡 |
| T+2 | 캐러셀 요약 | 인스타그램 |
| T+3 | 대비 카드 + 토론 포스트 | 트위터/X, 네이버 |

### 1개 영상에서 추출하는 콘텐츠

```
1 장편 영상 (15-20분)
    ├── 1 YouTube Short (Hook 60초)
    ├── 2-3 릴스/틱톡 (최강 인사이트, 30-90초)
    ├── 1 캐러셀 (인스타, 5-7페이지)
    ├── 3-4 포스트 카드 (인용, 수치, 대비)
    ├── 5-7 스토리 슬라이드
    └── 1 스레드 (트위터/X, 5-8트윗)
```

## 10. 업로드 전 체크리스트

### 영상 품질
- [ ] 오디오 명확, 왜곡 없음
- [ ] 이미지 선명, 픽셀깨짐 없음
- [ ] 자막/CC 정확
- [ ] 전환 부드러움
- [ ] 3초 이상 이미지 공백 없음
- [ ] 엔드 스크린에 링크 + 구독 버튼
- [ ] 영상 길이 정상 표시

### 메타데이터
- [ ] 제목 60자 이내, 수치 + 역설 포함
- [ ] 썸네일 1920x1080, 높은 대비, 모바일 가독성
- [ ] 설명에 타임스탬프, 출처, CTA, 해시태그
- [ ] 태그 완비
- [ ] 카테고리: 교육 또는 뉴스
- [ ] 언어: 한국어

### 크로스 프로모션
- [ ] 포스트 카드 시리즈 스케줄 등록
- [ ] YouTube Short 편집 완료
- [ ] 릴스/틱톡 준비 완료
- [ ] 각 플랫폼별 캡션 작성 완료
