# 🎴 Post Card Templates

## 1. Tổng quan

Post card là nội dung hình ảnh tĩnh hoặc carousel dùng cho Facebook, Instagram, TikTok, và các nền tảng khác. Mục đích:
- **Thu hút mới**: Kéo người xem từ mạng xã hội sang YouTube
- **Tóm tắt nội dung**: Đọng lại insight mạnh nhất từ video dài
- **Tạo thảo luận**: Kích thích bình luận và chia sẻ
- **Xây dựng thương hiệu**: Nhận diện ngôn ngữ và hình ảnh nhất quán

---

## 2. Các loại Post Card

### LOẠI 1: Quote Card (Trích dẫn đơn)

**Mục đích**: Đăng 1 quote mạnh từ video, kèm context ngắn.

**Layout**:
```
┌─────────────────────────────────┐
│  [LOGO KÊNH nhỏ — góc trên]    │
│                                 │
│                                 │
│     "QUOTE MẠNH NHẤT           │
│      TỪ VIDEO"                  │
│                                 │
│     — Ngữ cảnh ngắn 1 dòng     │
│                                 │
│                                 │
│  [Tên video — nhỏ, phía dưới]  │
│  [CTA: Xem full trên YouTube]  │
└─────────────────────────────────┘
```

**Quy tắc thiết kế**:
- Background: Tối (đen, navy đậm, hoặc gradient tối)
- Font quote: Bold, sans-serif, 36-48px
- Màu chữ quote: Trắng hoặc vàng nhấn (#F0C040)
- Dấu ngoặc kép lớn trang trí: Màu accent, opacity 20%
- Kích thước: 1080x1080px (Instagram) hoặc 1200x630px (Facebook)

**Copywriting**:
- Quote: Tối đa 25 từ (đọc được trong 3 giây)
- Caption: 2-3 câu mở rộng + hashtag + CTA

**Ví dụ**:
> Quote: *"Bạn không tiết kiệm — bạn đang trả phí cho sự yên tâm."*
> Context: *Khi lạm phát 8% nhưng lãi suất chỉ 4%*
> Caption: *"Mỗi năm bạn 'tiết kiệm', bạn thực chất đang mất 4% sức mua. Xem video đầy đủ trên YouTube để hiểu vì sao... 🔗 Link bio. #TàiChính #SựThậtTàn Nhẫn #TinhThức"*

### LOẠI 2: Data Bomb Card (Số liệu sốc)

**Mục đích**: Đăng 1 số liệu gây sốc + context.

**Layout**:
```
┌─────────────────────────────────┐
│                                 │
│         [SỐ LIỆU LỚN]         │
│            67%                  │
│                                 │
│   Người trẻ dưới 30 tuổi       │
│   không thể mua nhà             │
│                                 │
│   ─────────────────────         │
│   Nguồn: Ngân hàng Thế giới    │
│                                 │
│  [LOGO]    [Xem video đầy đủ]  │
└─────────────────────────────────┘
```

**Quy tắc thiết kế**:
- Số liệu: Font rất lớn, 96-120px, Bold, màu accent nổi bật
- Giải thích: Font nhỏ hơn, 20-24px, light weight
- Đường kẻ phân cách mảnh
- Nguồn: Font 14px, italic, opacity 70%
- Background: Gradient tối hoặc hình ảnh mờ liên quan

### LOẠI 3: Carousel (Chuỗi hình ảnh 3-7 trang)

**Mục đích**: Tóm tắt nội dung video thành chuỗi insight dễ lướt.

**Layout mỗi trang**:
```
Trang 1 (Cover):
┌───────────────────────────────┐
│                               │  
│   [TIÊU ĐỀ GÂY TÒ MÒ]      │
│                               │
│   [Hình ảnh/Icon minh họa]   │
│                               │
│   "Lướt để khám phá →"       │
│   [LOGO]                      │
└───────────────────────────────┘

Trang 2-6 (Nội dung):
┌───────────────────────────────┐
│   [SỐ/TÊN] — [TIÊU ĐỀ]     │
│   ─────────────────────       │
│                               │
│   [NỘI DUNG]                 │
│   • Insight 1                 │
│   • Insight 2                 │
│   • Số liệu quan trọng       │
│                               │
│   [PAGE INDICATOR: ●○○○○○]   │
│   [LOGO]                      │
└───────────────────────────────┘

Trang cuối (CTA):
┌───────────────────────────────┐
│                               │
│   Xem phân tích               │
│   đầy đủ trên YouTube         │
│                               │
│   [QR Code hoặc @tên kênh]   │
│                               │
│   Lưu bài • Chia sẻ          │
│   Follow để cập nhật          │
│   [LOGO]                      │
└───────────────────────────────┘
```

**Quy tắc thiết kế Carousel**:
- Tối đa **7 trang** (5 trang nội dung)
- Mỗi trang: **1 ý duy nhất**, đọc trong 5 giây
- Kích thước: 1080x1350px (tỉ lệ 4:5 cho Instagram)
- Thống nhất: cùng bảng màu, cùng font, cùng layout
- Page indicator: cho người xem biết vị trí

### LOẠI 4: Versus Card (So sánh đối lập)

**Mục đích**: So sánh hai thực tế đối lập để tạo nghịch lý.

**Layout**:
```
┌───────────────┬───────────────┐
│               │               │
│  [THỰC TẾ A]  │  [THỰC TẾ B]  │
│               │               │
│  Thu nhập     │  Người trẻ    │
│  tăng 40%     │  kiệt quệ    │
│               │               │
│  📈           │  💔           │
│               │               │
├───────────────┴───────────────┤
│   "Tại sao?" — Xem video      │
│   [LOGO]                       │
└───────────────────────────────┘
```

**Quy tắc thiết kế**:
- Chia đôi rõ ràng: bên trái sáng, bên phải tối (hoặc ngược lại)
- Font đối xứng
- Câu hỏi nối: ở giữa hoặc phía dưới

### LOẠI 5: Thread/Story Card (Chuỗi insight ngắn)

**Mục đích**: Dùng cho Instagram/Facebook Stories hoặc Twitter/X thread.

**Layout (9:16)**:
```
┌──────────────────────┐
│                      │
│  [LOGO nhỏ trên]     │
│                      │
│                      │
│  "INSIGHT MẠNH       │
│   TRONG 1-2 CÂU"     │
│                      │
│                      │
│  Swipe up ↑          │
│  để xem video        │
│                      │
└──────────────────────┘
```

**Kích thước**: 1080x1920px (9:16)

## 3. Bảng Màu & Typography

### Bảng màu chính

| Tên | Hex | Dùng cho |
|-----|-----|----------|
| **Deep Black** | `#0A0A0A` | Background chính |
| **Dark Navy** | `#0D1117` | Background phụ |
| **Ghost White** | `#E8E8E8` | Text chính |
| **Signal Gold** | `#F0C040` | Nhấn mạnh số liệu, quote |
| **Alert Red** | `#E53E3E` | Cảnh báo, con số tiêu cực |
| **Hope Teal** | `#38B2AC` | Giải pháp, tích cực |
| **Subtle Gray** | `#6B7280` | Text phụ, nguồn trích dẫn |

### Gradient

| Gradient | Dùng cho |
|----------|----------|
| `#0A0A0A → #1A1A2E` | Background card thường |
| `#0D1117 → #161B22` | Background carousel |
| `#F0C040 → #E53E3E` | Accent gradient (khẩn cấp) |
| `#38B2AC → #2D9CDB` | Accent gradient (hy vọng) |

### Typography

| Vai trò | Font | Weight | Size |
|---------|------|--------|------|
| Tiêu đề chính | Inter / Be Vietnam Pro | Bold (700) | 36-48px |
| Số liệu lớn | Inter / Montserrat | Extra Bold (800) | 72-120px |
| Nội dung | Inter / Be Vietnam Pro | Regular (400) | 18-24px |
| Nguồn/Caption | Inter | Light (300) | 14-16px |
| CTA | Inter | Semi Bold (600) | 18-20px |

## 4. Quy trình Sản xuất Post Card

### Từ Video → Post Card

1. **Trích xuất** 3-5 insight mạnh nhất từ kịch bản video
2. **Chọn loại card** phù hợp cho mỗi insight
3. **Viết copy**:
   - Quote: Tối đa 25 từ
   - Data: 1 số + 1 câu giải thích
   - Caption: 2-3 câu + hashtag
4. **Thiết kế** theo template và bảng màu
5. **Review** checklist
6. **Lên lịch đăng** theo content calendar

### Lịch đăng đề xuất (1 video = 3-5 post cards)

| Ngày | Post Card | Nền tảng | Ghi chú |
|------|-----------|----------|---------|
| Trước video 1 ngày | Teaser Card (Quote kích thích) | All | Tạo tò mò |
| Ngày đăng video | Data Bomb Card (số liệu sốc nhất) | All | Drive traffic |
| +1 ngày | Carousel (tóm tắt video) | Instagram | Recap cho ai chưa xem |
| +2 ngày | Versus Card (nghịch lý chính) | Facebook | Tạo thảo luận |
| +3 ngày | Quote Card (giải pháp) | All | Kết thúc chu kỳ |

## 5. Checklist Post Card

- [ ] **Đọc được trong 3 giây** — thông điệp rõ ràng ngay lập tức
- [ ] **Contrast đủ mạnh** — text nổi bật trên background
- [ ] **Logo kênh** có mặt (nhỏ, không chiếm focus)
- [ ] **CTA rõ ràng** — người xem biết phải làm gì tiếp
- [ ] **Kích thước đúng** chuẩn nền tảng target
- [ ] **Thống nhất brand** — cùng bảng màu, font, phong cách
- [ ] **Caption** đã viết kèm hashtag
- [ ] **Không lỗi chính tả**
- [ ] **Safe zone** — nội dung quan trọng không bị cắt ở mép
